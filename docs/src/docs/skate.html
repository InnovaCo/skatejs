skate (definition)

You define a component definition to the skate() function. The component name is used by the binding types to determine how to bind behaviour to the element. By default this conforms to the Custom Element Spec and corresponds to one of the following:

  • Tag name
  • is attribute (when the extends option is specified)

Native vs Polyfill

Underneath the hood, Skate will use native custom elements if the following criteria is met:

  • Your browser supports document.registerElement().
  • You have not specified the type option.
  • Your component has a hyphen in its name.

In all other instances Skate will use a MutationObserver to polyfill the same behaviour. In browsers that do not have native support for Mutation Observers, you must include a polyfill.

Though it's not slow by any means, you should always try and meet this criteria because using Native will always be faster and is synchronous out of the box.
