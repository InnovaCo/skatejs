options.extends

The name of the element that this component is extending. This is synonymous with the W3C Custom Element extends option.

When you specify the extends option, instead of your component name mapping to the tagName of an element, it maps to the value of the is attribute.

The default type conforms with the W3C spec and therefore uses this option. If you decide to use a custom type, that type may or may not use this option.
